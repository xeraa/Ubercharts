Ubercharts by xeraa

Charting API for Wicket, based on Highcharts

This library provides a simple integration of Highcharts in Wicket without any JavaScript meddling. This project requires Wicket 1.5 and builds upon Highcharts 2.1. For version 0.1 (Wicket 1.4 and Highcharts 2.0) see https://github.com/comsysto/Ubercharts
